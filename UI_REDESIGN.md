# YShop 咖啡点餐小程序 UI 重设计方案

> 参考：星巴克、瑞幸咖啡、Manner Coffee、Tims 天好咖啡等市场优秀产品
> 设计日期：2026-05-23

---

## 一、设计理念

### 品牌定位
**"你的精品咖啡生活"** — 打造温暖、专业、便捷的咖啡消费体验。

### 三大设计原则
| 原则 | 说明 |
|------|------|
| **温暖亲和** | 以咖啡色系为主调，降低冰冷感，拉近与用户的距离 |
| **高效直观** | 减少操作步骤，核心路径（选品→下单→支付）3步完成 |
| **精致细节** | 微交互、渐变、阴影层次、骨架屏加载，提升品质感 |

## 二、全新配色系统

### 主色调：咖啡暖色系

```
咖啡深棕          奶油白           焦糖金           拿铁灰
#4A2C1A (主色)    #FEF9F3 (背景)    #D4944A (强调)   #8B7355 (辅助)
```

### 完整色彩令牌

```
// ==== 品牌主色 ====
$color-primary:             #6B3A2E;    // 深咖啡（主按钮、选中态、品牌标识）
$color-primary-dark:        #4A2C1A;    // 深咖啡暗色（导航栏、强调文字）
$color-primary-light:       #FFF0E5;    // 浅咖啡（选中背景、标签）

// ==== 功能色 ====
$color-accent:              #D4944A;    // 焦糖金（促销标签、积分、热销标识）
$color-success:             #5B8C5A;    // 抹茶绿（成功、已支付）
$color-warning:             #E8A838;    // 蜂蜜黄（警告、待处理）
$color-error:               #C0392B;    // 红褐色（错误、退款）
$color-info:                #8B7355;    // 拿铁灰（提示信息）

// ==== 中性色 ====
$text-color-primary:        #2C1810;    // 最深 - 主标题
$text-color-regular:        #5D4037;    // 常规文字
$text-color-secondary:      #8D6E63;    // 辅助文字
$text-color-placeholder:    #BCAAA4;    // 占位符
$text-color-white:          #FFFFFF;    // 白色文字

// ==== 背景色 ====
$bg-color-page:             #FEF9F3;    // 页面背景（奶油白）
$bg-color-white:            #FFFFFF;    // 卡片背景
$bg-color-warm:             #FFF8F0;    // 暖色背景（头部区域）
$bg-color-card:             #FAFAF7;    // 卡片内背景
$bg-color-mask:             rgba(74, 44, 26, 0.5);  // 遮罩

// ==== 边框与阴影 ====
$border-color:              #E8D5C4;    // 浅咖啡边框
$border-color-light:        #F5EDE4;    // 更浅边框
$shadow-sm:                 0 2rpx 8rpx rgba(74, 44, 26, 0.06);
$shadow-base:               0 4rpx 16rpx rgba(74, 44, 26, 0.08);
$shadow-lg:                 0 8rpx 32rpx rgba(74, 44, 26, 0.12);
$shadow-button:             0 6rpx 20rpx rgba(107, 58, 46, 0.3);

// ==== 圆角 ====
$radius-sm:                 8rpx;       // 小圆角（标签、输入框）
$radius-base:               12rpx;      // 基础圆角（卡片、按钮）
$radius-lg:                 20rpx;      // 大圆角（面板、弹窗）
$radius-xl:                 28rpx;      // 超大圆角（底部栏、药丸按钮）
$radius-round:              999rpx;     // 圆形（头像、状态点）
```

### 对比：旧配色 vs 新配色

| 角色 | 旧值 | 新值 | 变化说明 |
|------|------|------|---------|
| 主色 | `#09b4f1` (冷青) | `#6B3A2E` (暖咖啡) | 从科技冷色调转变为咖啡暖色调 |
| 页面背景 | `#F1F8FA` (冷灰蓝) | `#FEF9F3` (奶油白) | 更温暖的基底色 |
| 主文字 | `#303133` (纯黑灰) | `#2C1810` (深棕) | 融入品牌色系的深色文字 |

## 五、核心组件设计

### 5.1 按钮
```
类型：主按钮 / 次按钮 / 文字按钮 / 危险按钮
主按钮：背景 $color-primary，白色文字，圆角 $radius-base
        阴影 $shadow-button
次按钮：背景 $color-primary-light，主色文字和边框
文字按钮：无背景，主色文字
危险按钮：背景 $color-error，白色文字
禁用态：opacity 0.5

尺寸：大(h=96rpx) / 中(h=72rpx) / 小(h=56rpx)
```

### 5.2 商品卡片
```
布局：横向：160rpx 缩略图 + 右侧信息区
样式：白色背景，圆角 $radius-base，阴影 $shadow-sm
缩略图：圆角 $radius-sm，object-fit cover
标题：$font-size-md，$font-weight-medium
描述：$font-size-sm，$text-color-secondary，最多 2 行
价格：$font-size-xl，$font-weight-bold，$text-color-primary
加购按钮：28rpx 圆形，背景 $color-primary
```

### 5.3 标签/Tag
```
圆角：$radius-sm
内边距：4rpx 12rpx
字体：$font-size-xs
类型：
  - 热销：红色背景 #FFF0F0 红色文字 #E74C3C
```
